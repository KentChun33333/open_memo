# extract-insights - Conversation Insights Extraction

Use when you want to extract key insights, decisions, or action items from a conversation history.

## Usage

```
/extract-insights
```

## What It Does

1. Analyzes the conversation history to identify:
   - Key decisions made
   - Action items and next steps
   - Important context or background
   - Technical work completed

2. Returns a structured summary

## Output Format

- Clear sections for decisions, actions, and context
- Timestamps when relevant
- Links to related files or resources

## Example Use Cases

- After building a dashboard: extract what was built and how to use it
- After debugging: extract the root cause and fix applied
- After planning: extract the agreement and next steps
